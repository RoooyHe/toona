# Product Overview

Toona is a Matrix-based task collaboration platform that combines instant messaging with kanban-style task management, inspired by Trello. Built on the Robrix Matrix client foundation, it integrates real-time team communication with visual task organization.

## Core Features

- Kanban board interface with drag-and-drop task cards
- Matrix protocol integration for end-to-end encrypted team communication
- Real-time multi-user collaboration with cross-platform sync
- Task management (todo/in-progress/done states)
- Chat rooms with rich messaging (text, images, replies, reactions, editing)
- User authentication and session persistence
- Avatar and media caching
- Space and room management
- Optional TSP (Trust Spanning Protocol) wallet integration

## Platform Support

Runs on macOS, Windows, Linux, Android, and iOS using a single Rust codebase.
